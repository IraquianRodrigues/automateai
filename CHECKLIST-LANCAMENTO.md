# ✅ Checklist de Lançamento - AutomateAI

Use este checklist antes de colocar o site no ar!

## 📋 Pré-Lançamento

### Informações Básicas
- [ ] WhatsApp atualizado em todos os componentes
- [ ] Email de contato configurado
- [ ] Telefone de contato atualizado
- [ ] Links de redes sociais configurados (Instagram, LinkedIn, Facebook)
- [ ] Logo da empresa adicionada
- [ ] Favicon personalizado

### Conteúdo
- [ ] Textos revisados (sem erros de português)
- [ ] Descrições dos serviços personalizadas
- [ ] Seção "Sobre Nós" com informações reais
- [ ] FAQ com perguntas relevantes
- [ ] Footer com informações corretas

### SEO e Metadata
- [ ] Title tags configurados
- [ ] Meta descriptions adicionadas
- [ ] Open Graph tags para redes sociais
- [ ] Sitemap.xml criado
- [ ] robots.txt configurado
- [ ] Google Analytics instalado (opcional)
- [ ] Google Search Console configurado (opcional)

### Funcionalidades
- [ ] Formulário de contato testado
- [ ] Links do WhatsApp funcionando
- [ ] Scroll suave entre seções funcionando
- [ ] Botões de CTA direcionando corretamente
- [ ] Links externos abrindo em nova aba

### Design e Responsividade
- [ ] Testado em desktop (Chrome, Firefox, Safari)
- [ ] Testado em mobile (iOS e Android)
- [ ] Testado em tablet
- [ ] Gradientes e cores alinhados com a marca
- [ ] Imagens otimizadas (WebP/AVIF se possível)
- [ ] Animações funcionando suavemente

### Performance
- [ ] Build de produção testado (`npm run build`)
- [ ] Lighthouse score > 90
- [ ] Imagens comprimidas
- [ ] Fontes otimizadas
- [ ] Sem erros no console

### Segurança
- [ ] HTTPS configurado
- [ ] Variáveis de ambiente configuradas (se aplicável)
- [ ] Rate limiting no formulário (recomendado)
- [ ] CSP configurado (recomendado)
